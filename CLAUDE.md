# Telegram → Notes (Claude-powered)

## Что это
Система для автоматической обработки заметок из Telegram.
Я (Claude) получаю новые сообщения через Scheduled Task, сортирую их по категориям и записываю в Markdown-файлы с комментариями.

## Как работает

1. **Telegram Bot** (`bot.py`) — слушает сообщения, сохраняет в `inbox.json`
2. **Scheduled Task** — запускает меня (Claude) по расписанию или вручную
3. **Я** — читаю `inbox.json`, обрабатываю каждую заметку, пишу в нужный файл в `notes/`, очищаю inbox

## Структура проекта

```
telegramObsidian/
  CLAUDE.md              # этот файл — главная инструкция
  bot.py                 # Telegram bot (получает сообщения)
  inbox.json             # буфер входящих (необработанных) заметок
  .env                   # TELEGRAM_TOKEN (не коммитить)
  notes/                 # папка с заметками по категориям
    health.md
    ideas.md
    shopping.md
    finance.md
    books.md
    ...                  # новые категории создаю сам по необходимости
```

## Формат inbox.json

```json
[
  {
    "id": "uuid",
    "text": "Head & Shoulders шампунь от перхоти",
    "timestamp": "2026-03-30T10:00:00",
    "processed": false
  }
]
```

## Формат заметки в notes/*.md

```markdown
## 2026-03-30

- **Head & Shoulders** — шампунь против перхоти
  > Активное вещество: пиритион цинка. Использовать 2–3 раза в неделю.
```

## Правила категоризации

- Если в сообщении явно указана категория (например, "здоровье: ...") — использую её
- Иначе — определяю сам по смыслу
- Если категория новая — создаю новый файл в `notes/`
- Если неоднозначно — кладу в `ideas.md` с пометкой

## Моя задача при обработке

1. Прочитать `inbox.json`
2. Для каждого необработанного сообщения:
   - Определить категорию
   - Дописать в `notes/<category>.md` с датой и комментарием
   - Отметить `processed: true`
3. Удалить обработанные из `inbox.json`

## Статус разработки

- [ ] `bot.py` — Telegram bot
- [ ] `.env` — токен (пользователь добавляет сам)
- [ ] `inbox.json` — инициализирован пустым
- [ ] Scheduled Task настроен

## Переменные окружения (.env)

```
TELEGRAM_TOKEN=your_token_here
ALLOWED_USER_ID=your_telegram_user_id
```

`ALLOWED_USER_ID` — чтобы бот принимал сообщения только от тебя.
